# 콘티노트(ContiNote) — 로고 제작용 PRD / 브랜드 브리프

> 목적: 로고(앱 아이콘·파비콘·워드마크·마스코트) 제작 시 디자이너 또는 AI에게 그대로 전달하는 브리프.
> 작성일: 2026-06-07 · 출처: 실제 코드(`globals.css`, `manifest.json`, `icon.svg`) 및 PPT 테마 설정에서 확인.

---

## 1. 한 줄 정의
**콘티노트(ContiNote)** — 악보 사진/PDF를 올리면 **교회 찬양팀 예배 화면용 PPT가 5분 안에 완성**되는 AI 콘티 메이커.

- 표기: **콘티노트 / ContiNote**
- 한 줄 비전: "토요일 밤 PPT 만들기 1~2시간 → 5분"

## 2. 슬로건 / 태그라인 (현재 사용 중)
- 메인: **"악보를 콘티 가사로, 클릭 한 번에"**
- 서브: "찬양팀·예배 사역자를 위한 AI 콘티 메이커"

## 3. 타겟 사용자
- 핵심: **찬양팀장 / 음악팀 리더** (30~50대, 교회 자원봉사자)
- 보조: 방송팀(PPT 송출 담당)
- IT 수준: 카톡·유튜브·한컴 가능, 코딩/디자인 경험 없음 → **친근·쉬움**이 중요
- 사용 주기: 주 1회(주말 예배 준비)

## 4. 핵심 가치 · 차별점
- **시간 절감** — OCR→가사추출→콘티편집→PPT를 한 흐름으로 (30분→5분)
- **초보자 친화** — 설명서 없이 바로 사용 / 무료
- **한국 교회 맞춤** — 예배실 스크린·찬양팀 워크플로우에 최적화
- vs ProPresenter/EasyWorship: 무료 + 비전공자 친화 + 한국 찬양팀 맞춤

## 5. 주요 기능 (요약)
악보 OCR(최대 12장) · AI 가사 섹션 분류(Verse/Chorus/Bridge…) · 드래그앤드롭 콘티 편집 · PPT 6테마·4폰트 생성 · 슬라이드 상/중/하 정렬 · 다중 내보내기(PPTX/TXT/DOCX/OpenSong) · 곡 라이브러리 · 구글 로그인 동기화 · PWA(오프라인).

## 6. 브랜드 성격 / 톤앤매너 키워드 ★로고 핵심
- **따뜻함 · 진정성** (한국 교회 정서)
- **친근함** (위협적이지 않은, 도와주는 친구 같은 AI)
- **신뢰감 · 차분함** (과장 없는, 예배에 어울리는 절제)
- **현대성 + 전통의 균형** (산세리프 + 명조)
- 한 마디로: **"예배 준비를 도와주는 따뜻한 친구"**

## 7. 컬러 방향 (현재 팔레트)
| 역할 | HEX | 비고 |
|------|-----|------|
| 메인 강조 (테라코타) | `#C77A56` | PWA theme_color, 브랜드 주색 |
| 강조 강(hover) | `#A85F3E` | |
| 종이 배경(크림) | `#FAF5EC` / `#FFFDF8` | paper 테마·PWA 배경 |
| 텍스트(잉크) | `#1F1B16` | 진한 차콜 |
| 마스코트 황금색 | `#F7CA52` | 병아리 몸통 |
| 마스코트 배경 | `#FFF3D4` | 밝은 노랑 |

> 핵심 조합: **테라코타(#C77A56) × 따뜻한 크림 × 병아리 황금색**. 교회 목재/온벽돌 톤.
> (현대 옵션 'Wanted 테마'는 흰 배경 + 일렉트릭 블루 `#0066FF` — 보조 방향)

## 8. 폰트 방향
- 디스플레이: **Wanted Sans**(현대 산세리프) 또는 **Noto Serif KR**(명조, 신앙 감성)
- 본문: **Pretendard**
- 워드마크: 명조(신뢰·전통) / 산세리프(현대·앱) 둘 다 시안 권장

## 9. 로고에 담을 상징 / 모티프 후보
- **병아리 마스코트** ← 현재 사용 중. 황금색 몸 + 테라코타 부리/눈썹, 연분홍 볼 → "친근한 도우미"
- 음표 / 악보 / 슬라이드(화면) 모티프
- 노트(콘티 = 콘티노트의 "노트") — 펼친 노트·줄노트
- 십자가는 **은은하게**만 (예배 맥락이되 과한 종교 심볼은 지양)

## 10. 피해야 할 것 (Avoid)
- 무겁고 권위적인 전통 교회 이미지 (타겟은 실무 자원봉사자, 친근함이 핵심)
- 차가운 테크/AI 느낌의 푸른 네온·로봇
- 과한 종교적 장식 (실용 도구임을 잊지 말 것)
- 이모지·복잡한 디테일 (크로스플랫폼 일관성 위해 본 서비스는 이모지 미사용)
- 회피 단어 톤: "워십/프레젠테이션/생성/에러/고객님" 대신 "콘티/가사/슬라이드/만들기/함께 사역"

## 11. 사용처 & 기존 자산
- 앱 아이콘, 파비콘, apple-touch-icon, PWA 아이콘(192/512 + maskable)
- 웹 헤더 워드마크, 로딩/빈 상태의 마스코트
- 기존 자산: `public/icon.svg`(병아리), `favicon.ico`, `icon-192/512.png`, `apple-touch-icon.png`
- 마스코트 상태 PNG: `done/idle/reading/listening.png` (히어로·로딩·대기 연출)
- manifest: theme_color `#C77A56`, background `#FFFDF8`
