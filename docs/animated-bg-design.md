# 움직이는 홀리 배경 — 디자인 스펙 (v4)

`scripts/gen-holy-bg.mjs`가 이 스펙대로 GIF를 생성한다. 새 배경을 추가할 때도 이 문서의
공통 원칙을 지킨다. (사용자 피드백으로 다듬어진 규칙들 — 어기면 "짜친다" 소리 듣는다.)

## 공통 원칙 (전 테마 적용)

1. **가사가 주인공** — 화면 정중앙(세로 밴드)은 항상 잔잔하게. 효과는 가장자리·상단·하단에.
   모든 배경은 어두운 톤 → PPT 글자는 흰색 고정.
2. **깜빡임 금지** — 별 반짝임·점멸 효과 금지(눈 아픔). 허용되는 움직임은
   "떠다님 / 숨 쉼 / 흐름 / 가라앉음"뿐.
3. **기계적 펄스 금지** — 단일 sin으로 크기가 커졌다 작아지는 원은 단조롭다.
   숨 쉬는 느낌이 필요하면 **불투명도만** 미세하게(±0.05 수준), 또는
   다중 사인 합성(`0.6sin(t)+0.3sin(2t+1.3)+0.2sin(3t+2.7)`)으로 유기적으로.
4. **끊김 없는 루프** — 모든 시간 파라미터는 t(0..2π)의 정수배 sin/cos이거나,
   진행도 p=frac(t/2π) 기반 + 양 끝 페이드(sin(πp))로 4초에 정확히 한 바퀴.
5. **렌더 품질** — 1280×720, 32프레임, 8fps. ffmpeg palettegen(stats_mode=diff)+
   sierra2_4a 디더. 단색 계열은 160색, 다색·복합 그라데이션은 200색.
6. **SVG 함정** — 그라데이션 stop의 currentColor는 검정이 된다 → 색마다 그라데이션 생성.

## 패밀리 1 — 글로우+보케 6종 (새벽 구성 고정, 색 계열만 교체)

구성: 어두운 대각 그라데이션 + 우상단 큰 글로우(좌우 ±2.5% 이동, 불투명도 호흡)
+ 좌하단 보조 글로우 + 가장자리 보케 9개(±10~28px 떠다님, 위상 제각각).

## 패밀리 2 — 빛내림 (light)

칠흑 배경, 광원은 가운데보다 왼쪽(43%)·**고정 크기**(r 8%, 펄스 없음).
광선 13개: 길이 0.5~1.35H 제각각, 폭 20~46px(얇게), 좌우 미세 스웨이,
밝기만 위상 다르게 호흡(0.12~0.32). 끝은 82%에서 완전 페이드(사각형 끝 금지). 블러 6.

## 패밀리 3 — 신규 6종 (각자 고유 구성)

### wave 물결 (잔잔한 수면)
- 수평선 y=62%. 위=짙은 밤하늘+우측(68%) 달무리(크기 고정, 불투명도만 ±0.05).
- 아래=물: 달 아래로 수면 반사 줄기 9개(가로로 긴 타원, 아래로 갈수록 넓고 옅게).
  각 줄기는 좌우 스웨이(4~16px, 위상 제각각)+밝기 미세 호흡 → "물에 비친 빛이 일렁".
- 용도: 세례식, "생수의 강" 찬양.

### mist 안개 (새벽 묵상)
- 청회색 베이스. 안개 4개 층(y 20/45/70/88%) — 층마다 폭넓은 타원 3개가
  좌우로 천천히 스웨이(±25~40px, 층마다 속도·위상 다름), 불투명도 0.06~0.14.
- 상단에 차가운 새벽빛이 옅게 비침. 전체적으로 가장 절제된 배경.

### candle 촛불 (따뜻한 기도)
- 거의 검은 따뜻한 배경 + 하단 중앙 촛불빛 돔.
- 일렁임은 다중 사인 합성(3화음) — 불투명도·중심 좌우 6px만 흔들리고 크기는 고정.
- 작은 밝은 심지광 + 넓은 온기 글로우 2겹. 성탄 전야·조용한 기도회.

### grace 빛가루 (내리는 은혜)
- 보랏빛 새벽 배경 + 상단 옅은 빛.
- 빛 입자 26개가 **아주 천천히 가라앉음**(4초에 80~140px) — 각 입자는
  진행도 p마다 sin(πp)로 스르륵 나타났다 사라져서 루프가 안 보임. 좌우 미세 드리프트.
- 깜빡임이 아니라 "낙하+페이드"라 눈 안 아픔. 성탄절·축복 찬양.

### aurora 오로라 (밤하늘 물결)
- 거의 검은 남색 하늘 + **고정 밝기** 잔별 30개(애니메이션 없음 — 깜빡임 금지 원칙).
- 상단 1/3에 오로라 리본 3개(틸·그린·퍼플): 24점 폴리곤 띠의 윗선이
  사인파로 천천히 물결침(리본마다 파수·위상·두께 다름), 세로 그라데이션+블러 12.
- 송구영신·밤 집회.

### crosslight 십자가빛 (역광)
- 거의 검은 배경. 좌측 28% 지점에 따뜻한 역광 글로우(크기 고정, 불투명도만 호흡).
- 글로우 위에 십자가 실루엣(검정) — 빛을 등진 형상. 가사 영역(중앙·우측)은 어둠 유지.
- 고난주간·부활절.
