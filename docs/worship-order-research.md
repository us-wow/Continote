## 공통 골격 (최빈 순서)

```
입례송 → 예배의 부름 → (송영) → 찬송 → 신앙고백(사도신경) → 성시교독
→ 찬송 → 기도 → (성도의 교제) → 성경봉독 → 찬양 → 설교
→ 찬송 → 봉헌(+봉헌기도) → (교회소식) → 찬송 → 축도
```

교단별 특색: 감리교는 교독문·송영이 풍성한 예전형 / 침례·순복음은 항목이 짧고
독자 표기(예배선언·말씀증거·기도와 결신) / 성결은 중간형.

## 설계 결론 (예배 순서 빌더)

1. **블록 기본 이름 = 최빈 표기**: 입례송 / 예배의 부름 / 송영 / 찬송 / 신앙고백(사도신경) /
   성시교독 / 기도 / 성도의 교제 / 성경봉독 / 찬양 / 설교 / 봉헌 / 봉헌기도 / 교회소식 /
   주기도문 / 참회기도 / 축도 (+성찬식은 옵션 블록)
2. **블록 이름은 반드시 수정 가능하게** — 같은 항목을 12교회가 거의 12가지로 부른다.
   특히 "예배의 부름"은 표본 안에서만 6가지 표기. 기본값은 주고, 표기는 교회의 것.
3. **기본 템플릿은 위 공통 골격**으로 제공하되, 블록 추가·삭제·드래그로 1회 조립
   → "우리 교회 템플릿" 저장.
4. **교독문·성경 본문은 자동 내장 금지**(저작권: 대한성서공회·찬송가공회) — 사용자
   붙여넣기. 사도신경·주기도문 텍스트는 고정문으로 내장 가능.
5. 광고(성도의 교제) 위치가 교회마다 앞/뒤로 갈리므로 위치 고정 금지.
