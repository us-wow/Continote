# 콘티노트 사용법

찬양팀·예배 사역자를 위한 AI 콘티 메이커. 악보를 올리면 가사만 추출해서 PPT 슬라이드까지 한 번에 만듭니다.

---

## 빠른 시작 (5단계)

1. **악보 올리기** — JPG/PNG/PDF 드래그
2. **가사 추출하기** 버튼 클릭
3. 좌측 결과에서 **곡 제목 + 섹션 카드 클릭** → 우측 콘티에 추가
4. 콘티 편집창에서 **Enter로 슬라이드 분리**
5. **PPT 다운로드 (.pptx)** 클릭

---

## 1. 악보 업로드

- 지원 형식: **JPG · PNG · PDF**
- 드래그&드롭 또는 클릭해서 파일 선택
- 한 번에 **최대 12개** 업로드 가능
- PDF는 자동으로 페이지별 이미지 변환됨

### 정확도 우선 토글
| 모드 | 동작 | 언제 쓰나 |
|------|------|----------|
| OFF (기본) | Flash 모델 + 자동 화질 | 깔끔한 악보, 빠른 추출 원할 때 |
| ON | Flash 모델 + 더 신중한 prompt + PDF 고화질(scale 2) | 흐릿한 PDF, 가사가 빠짐없이 정확해야 할 때 |

→ 정확도 ON은 살짝 느리지만 OCR 정확도가 올라갑니다.

### 직접 가사 붙여넣기 토글
악보 없이 텍스트로 바로 시작하고 싶을 때. 토글 켜고 가사 텍스트 입력 → 추출하기.

---

## 2. 추출된 곡 (좌측 결과 패널)

### 곡 제목 카드
- **클릭** → 우측 콘티에 제목 추가 (PPT 첫 슬라이드가 됨)
- **✎** → 제목 수정
- **✕** → 곡 통째로 삭제 (콘티에 추가된 블록도 함께)

### 섹션 카드 (Verse / 후렴 / Bridge / Pre-Chorus 등)
- **클릭** → 우측 콘티에 섹션 추가
- **↑ ↓** → 같은 곡 안에서 섹션 순서 바꾸기
- **✎** → 섹션 가사/타입 수정

### 라벨 자동 갱신
같은 type(verse 등)이 여러 개면 자동으로 "Verse 1", "Verse 2"로 번호가 매겨집니다. 순서를 바꾸면 라벨도 자동으로 따라옵니다.

---

## 3. 콘티 편집 (우측 패널)

### 블록 종류
- **곡 제목** — 큰 헤더, PPT에서 첫 슬라이드
- **섹션** — 가사 본문, 클릭해서 인라인 편집 가능
- **빈 줄** — 시각적 분리
- **─ 슬라이드 구분 ─** — 명시적 슬라이드 분리자

### 블록 동작
- **↑ ↓** — 블록 위/아래 이동
- **제거 / ✕** — 블록 삭제

### 가사 편집
섹션 본문을 직접 클릭하면 contentEditable로 편집됩니다. **수정 후 다른 곳 클릭하면 반영**.

### 헤더 버튼
- **+ 슬라이드 구분** — 콘티 끝에 슬라이드 분리자 추가 (↑↓로 위치 조정)
- **전체 비우기** — 콘티 전체 초기화

---

## 4. 슬라이드 분리 규칙 ⭐

**4가지 방법** (자유롭게 조합):

### 방법 1: Enter 두 번 (빈 줄) — 가장 직관적
가사 안에서 Enter를 두 번 누르면 빈 줄이 생기고, 그 위치에서 PPT 슬라이드가 분리됩니다.
**백스페이스로 빈 줄을 지우면 즉시 다시 합쳐집니다.**

### 방법 2: 섹션 경계 — 자동
Verse 다음 후렴이면 자동으로 다른 슬라이드로 분리됩니다.

### 방법 3: + 슬라이드 구분 버튼
콘티 헤더의 [+ 슬라이드 구분] 버튼으로 명시적 분리자를 추가. 블록 사이의 위치 조정 가능.

### 방법 4: 합치기 (분리 안 하기)
Enter 한 번(줄바꿈)만 하면 같은 슬라이드 안 여러 줄. 4줄까지 가능.

---

## 5. PPT 제작 (4번 영역)

### 폰트 선택 (4종)
- **본명조 Pro** (Noto Serif KR) — 기본, 한국 CCM에서 가장 모던
- **나눔명조** — 클래식 명조
- **나눔스퀘어** — 산세리프, 깔끔
- **본고딕** (Noto Sans KR) — 시스템 호환성 좋음

### 미리보기 카드
각 슬라이드 카드에 표시되는 정보:
- 슬라이드 번호 · 줄 수 · 폰트 사이즈
- 가사 미리보기
- ⚠️ 한도 초과 시 빨간색 + 분리 안내

### 한도 표 (자동 적용)
| 줄 수 | 폰트 사이즈 | 줄당 글자 한도(띄어쓰기 포함) |
|------|------------|--------------------|
| 1줄 | **64pt** | 17자 |
| 2줄 | **54pt** | 21자 |
| 3줄 | **44pt** | 26자 |
| 4줄 | **36pt** | 32자 |
| 5줄 이상 | 🔴 분리 필요 | — |

### 자동 안전망
한도 통과 후에도 폰트마다 미세하게 텍스트 박스를 넘는 경우, PowerPoint가 **자동으로 폰트를 살짝 축소(shrink-to-fit)**해서 한 줄에 맞춥니다.

### 다운로드
- 폰트 드롭다운 선택 후 **PPT 다운로드 (.pptx)** 클릭
- 한도 초과 슬라이드가 있으면 다운로드가 차단되고 토스트 안내 → 콘티에서 분리 후 재시도

### 슬라이드 디자인
- **검정 배경 + 흰 글자**
- 16:9 와이드 (1920×1080)
- **가운데 정렬**

---

## 6. 다른 다운로드

### TXT 다운로드
콘티 텍스트만 저장. 곡 제목은 `━━━ 제목 ━━━`로 강조.

### DOCX 다운로드
워드 문서. 곡 제목은 가운데 정렬 헤딩으로 변환.

### 클립보드 복사
콘티 전체 텍스트를 클립보드에 복사.

---

## 권장 사용 흐름 (예시)

### 시나리오: 한 곡 콘티 만들기
1. 악보 PDF 업로드 → 가사 추출
2. 곡 제목 카드 클릭 (콘티에 제목 추가됨)
3. 후렴 카드 클릭 (가사 추가)
4. Verse 1 카드 클릭
5. 후렴 카드 다시 클릭 (반복용)
6. Verse 2 카드 클릭
7. 후렴 카드 클릭 (마지막 반복)
8. 가사 안에서 너무 긴 줄은 Enter 두 번으로 분리
9. 4번 영역 미리보기 확인 — 빨강 없으면 OK
10. **본명조 Pro** 선택 후 **PPT 다운로드**

### 시나리오: 여러 곡 메들리
- 곡 1 제목 → 곡 1 섹션들 → 곡 2 제목 → 곡 2 섹션들 ...
- 각 곡 제목이 자동으로 별도 슬라이드 (표지 역할)

## 자주 묻는 질문

**Q: PPT 한 줄이 박스를 넘어서 잘려요.**  
A: PowerPoint의 shrink-to-fit이 자동으로 살짝 축소해줍니다. 그래도 너무 길면 빨강 알림 → Enter 두 번으로 분리하세요.

**Q: 가사가 빠지거나 잘못 추출됐어요.**  
A: 정확도 우선 토글을 켜고 다시 추출해보세요. 그래도 안 되면 ✎ 버튼으로 직접 수정 가능.

**Q: 곡 카드의 ✕로 곡을 지웠는데 콘티 편집창에도 영향?**  
A: 네, 그 곡과 연결된 콘티 블록도 함께 사라집니다. 미리 confirm으로 확인합니다.

**Q: 한 섹션 안 가사가 너무 길어요.**  
A: contentEditable로 직접 편집해서 줄바꿈 추가하거나, Enter 두 번으로 슬라이드 분리하세요.

**Q: PPT 폰트가 다른 걸로 보여요.**  
A: PPT 폰트는 시스템에 그 폰트가 설치돼 있어야 정확히 표시됩니다. 본명조 Pro(Noto Serif KR)가 가장 호환성이 좋아 추천.

**Q: PPT 작업 영역이 우측 콘티 편집창에 가려져요.**  
A: PPT 영역은 좌측 컬럼 1·2·4 흐름으로 배치돼 있어서 가려지지 않습니다. (콘티 편집창은 우측 sticky)

---

## 단축키 / 팁

- 콘티 편집창 가사 안에서 **Enter** = 줄바꿈, **Enter 두 번** = 슬라이드 분리
- **백스페이스로 빈 줄 지우기** = 슬라이드 합치기 (즉시 반영)
- 곡 제목 클릭 → **이미 추가됨** 표시면 더 이상 안 들어감 (중복 방지)
- 섹션 카드 클릭 → 매번 추가됨 (후렴 반복 등 자연스러움)

---

## 문제 해결

- **PPT 다운로드 시 오류** → 새로고침 후 재시도. 그래도 안 되면 콘티 영역에 5줄 이상 슬라이드 있는지 확인
- **가사 추출 시 JSON 오류** → 정확도 모드 ON으로 전환 (Pro 모델 thinking 이슈는 v3에서 해결됨)
- **모바일에서 마스코트 겹침** → 캐시 비우고 새로고침
